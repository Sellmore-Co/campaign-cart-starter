Background Colors

Core background color utilities that can be applied to any element to establish your site's visual hierarchy. These classes set only the background color property, allowing you to build layouts with consistent coloring without affecting other styling properties.

The system includes primary, secondary, and tertiary colors with light variations, plus accent and complementary colors to create visual interest and guide user attention. Use these background colors to create depth, establish sections, and maintain brand consistency throughout your interface.

COLORS & BACKGROUNDS

Section Gradients

Predefined gradient elements designed to create smooth color transitions within sections. These gradient components add visual depth and can guide user attention through your layouts.

Place these gradient elements inside containers with relative positioning to ensure proper layering. They can be positioned at the top, bottom, left, or right of a section to create directional color flows that enhance your design without requiring custom CSS for each implementation.

Custom Gradients

An attribute-based system for applying gradient backgrounds without creating additional classes. Simply add the gradient attribute to any element to apply a predefined gradient.

Icons

Icon classes provide standardized sizing and positioning for SVG, font, and image-based icons. The base icon size is 1.125em × 1.125em, allowing icons to scale proportionally with surrounding text.

Choose between em-based sizing (scales with parent font size), rem-based sizing (consistent regardless of context), or auto-width options for image-based icons. For maximum flexibility, wrap icons with the icon-wrapper class before converting to components to maintain proper sizing and alignment.

image-wrapper

image-wrapper is a generic base class that has overflow hidden, position relative, and width 100%. This is useful to avoid creating a new class for standard-sized images in columns. It is also aligned top to avoid stretching for the whole column (you can change that by placing it in a flex parent). Use a combo class to give it full height.
If you have images with unique sizes and positioning, create a new class for it.

It has combo classes to control its aspect ratio or set it to contain. You can add more or modify them. Default:
